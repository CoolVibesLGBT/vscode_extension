# CoolVibes for VS Code

CoolVibes for VS Code brings **CoolVibes**, the **LGBTQIA+ social network and gay community platform**, directly into your editor. Open CoolVibes in a dedicated sidebar or panel and stay connected to your community without leaving VS Code.

This extension packages the existing CoolVibes web experience for Visual Studio Code, making it easier to access conversations, stories, discovery surfaces, notifications, and community activity from a familiar in-editor workflow.

## Why Install CoolVibes for VS Code

- Open CoolVibes from the VS Code Activity Bar
- Launch a larger dedicated panel with `CoolVibes: Open Panel`
- Use the existing CoolVibes web experience inside a VS Code webview
- Stay close to your LGBTQIA+ social network while you work
- Keep access to a consistent, multilingual interface adapted for the extension environment

## What Is CoolVibes

CoolVibes is an **LGBTQIA+ social network**, **gay social network**, and inclusive community platform designed to help people connect, share, and build meaningful relationships. It brings together social interaction, messaging, stories, discovery, and community engagement in one experience built for queer communities worldwide.

If you are looking for a **gay community app**, **LGBT social platform**, or **queer social network** that is also accessible from your development environment, CoolVibes for VS Code provides a direct bridge between the CoolVibes platform and Visual Studio Code.

## Key Features

- Fast access from the sidebar for always-available community browsing
- Dedicated panel mode for a larger and more focused experience
- Support for the core CoolVibes web interface inside VS Code
- External link handling integrated with the VS Code environment
- Asset and runtime adjustments tailored for webview compatibility

## Getting Started

1. Install **CoolVibes** from the Visual Studio Code Marketplace.
2. Open the **CoolVibes** icon from the Activity Bar.
3. Or run `CoolVibes: Open Panel` from the Command Palette.
4. Sign in with your CoolVibes account to continue inside VS Code.

## Command

| Command | Description |
| --- | --- |
| `CoolVibes: Open Panel` | Opens CoolVibes in a dedicated editor panel. |

## Who This Extension Is For

CoolVibes for VS Code is built for developers, creators, and community members who want direct access to a modern **LGBTQIA+ social network**, **gay social app**, and inclusive community experience while working in Visual Studio Code.

## Install from Source

```bash
npm install
npm run build
```

Then press `F5` in VS Code to launch an Extension Development Host and test the extension locally.
